# claude-plugins-frasalvi
Marketplace for my custom set of plugins for Claude Code

## Plugins

This repository contains the following plugins:

- **research-companion**: A plugin for brainstorming, evaluating, and deciding on research directions.
- **project-oversight**: A plugin for overseeing and managing projects, breaking them down into manageable milestones and tasks.
- **pr-review-triage**: A plugin for triaging and reviewing pull requests.
